# deliciousBoard
API기반 게시판 프로그램<br>

### 기술스택<br>

JDK : openjdk17<br> 

RDMS : MySQL<br>

Spring Boot : springframework.boot (version. 2.6.7) <br>

SQL Mapper : MyBatis <br>

Build Tool : Gradle <br>

Test : JUnit5, AssertJ <br>

### API 목록<br>

1. 게시물 생성 <br>
2. 게시물 상세 조회<br> 
3. 게시물 목록 조회 <br>
4. 게시물 삭제 <br>
5. 게시물 검색 (단어) <br>
6. 특정기간 게시물 조회 <br>
